# 数据类型

## 1、数字类型

python有三种数字类型:int、float、complex(复数)

> **complex复数**
>
> - 复数以“ j”表示为虚部

- 从复数x提取实部和虚部，可使用`x.real`和`x.imag`

```python
x = 3+4j # 复数

print(x.real) # 提取实部
print(x.imag) # 提取虚部
```

- 类型转换：

```python
x = 1 # int
y = 2.8 # float
z = 1j # complex

#convert from int to float:
a = float(x)
#convert from float to int:
b = int(y)
#convert from int to complex:
c = complex(x)
```
